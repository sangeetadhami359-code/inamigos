# InAmigos Foundation Website

Welcome to the official repository for the **InAmigos Foundation** website. 

InAmigos Foundation is a Section 8 licensed non-profit NGO based in Bilaspur, Chhattisgarh, India, dedicated to driving social change through diverse community initiatives.

## 🚀 Live Preview
Explore the website: [InAmigos Foundation](https://sangeetadhami359-code.github.io/inamigos/) *(Note: replace with actual GitHub Pages link if hosted)*

## 🌍 Our Key Initiatives
We operate multiple targeted social initiatives across India:
- **Project SEVA**: Distributing warm meals, blankets, and providing hospital care.
- **Project Bachpanshala**: Supporting primary education, digital literacy, and creative workshops.
- **Project Jeev**: Rescuing street animals, providing primary care, and managing feeding drives.
- **Project Udaan**: Empowering women through skill-building, financial independence, and mentorship.
- **Project Prakriti**: Driving environmental conservation, large-scale plantation drives, and plastic-free campaigns.
- **Project Vikas**: Mentoring young professionals through virtual internships and practical exposure.

## 💻 Tech Stack
The website is built entirely with modern, lightweight, and performant core web technologies to ensure fast load times and universal accessibility:
- **HTML5**: Semantic and accessible structure.
- **CSS3 / Vanilla CSS**: Custom responsive styling, fluid flex/grid layouts, smooth micro-animations, and dynamic light/dark mode toggling.
- **JavaScript (ES6+)**: Interactive modals, carousel galleries, and state management.
- **FormSubmit API**: Secure, backend-free contact and volunteer application form processing.

## 🛠️ Features
- **Fully Responsive Design**: Optimized for mobile, tablet, and desktop viewing.
- **Interactive Modals**: Seamlessly read blogs, event details, and volunteer forms without leaving the page.
- **Dynamic Theming**: Integrated Dark/Light mode toggle for improved accessibility and reading comfort.
- **Automated Email Routing**: "Contact Us" and "Join as Volunteer" forms automatically dispatch submissions securely without exposing email logic.
- **LeafletJS Map Integration**: Interactive location mapping for the main office.

## 🤝 How to Run Locally
1. Clone this repository:
   ```bash
   git clone https://github.com/sangeetadhami359-code/inamigos.git
   ```
2. Navigate into the project folder:
   ```bash
   cd inamigos
   ```
3. Run a local development server (e.g., using `npx serve` or Live Server):
   ```bash
   npx serve -l 3000
   ```
4. Open `http://localhost:3000` in your browser.
